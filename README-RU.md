# Paseka

Paseka — локальный CLI-рантайм для запуска AI-агентов внутри git-репозитория. Проект помогает соло-разработчику инициализировать колонию, запускать пчёл, хранить состояние рантайма и обмениваться событиями через NATS и JetStream.

## Что умеет проект

- Инициализировать конфигурацию колонии в `.paseka/`.
- Запускать рантайм улья для репозитория.
- Отправлять одноразовые задачи пчёлам из CLI.
- Открывать интерактивные сессии с участием разработчика.
- Публиковать, проигрывать и проверять доменные события.
- Запускать локальную веб-консоль Queen Console.

## Как запустить

1. Склонировать репозиторий:

```
git clone https://github.com/russ-p/paseka.git
cd paseka
```

2. Установить зависимости и собрать бинарник:

```
go mod download
go build -o paseka ./cmd/paseka
```

3. Инициализировать колонию и запустить рантайм:

```
./paseka init
./paseka run
```

4. Запустить локальную веб-консоль Queen Console:

```
./paseka console
```

Откройте http://127.0.0.1:8787 в браузере. Queen Console пока не требует аутентификацию, поэтому оставляйте её доступной только на localhost или другом доверенном интерфейсе. Используйте `--addr`, только если понимаете риск внешнего доступа.
